
ABOUT
=======
Function : This program displays information about all block devices in a nice GUI.
Os       : Linux/KDE5 or KDE4
License  : Gpl
Version  : 0.9

BDGUI displays information about hardisks,loopback devices, iscsi, llvm,linux raid,btrfs raid,fstab, statistics etc...and you can also mount devices.

For more information, please look in the Docs folder
